D3 Charting Experiments
============================

- `npm install -g grunt`
- `npm install -g bower`
- `npm install`
- `bower install`
- `grunt`
- Visit [localhost:9000/line.html](localhost:9000/line.html) or [localhost:9000/bar.html](localhost:9000/bar.html)
